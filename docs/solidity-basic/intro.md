# Solidity简介

Solidity是一门专门用来编写智能合约的静态类型语言。它被广泛应用在各种公共或者私有区块链平台(以太坊，Polygon, BSC等)。它是我们入门智能合约的首选语言，目前围绕着Solidity已经产生了许许多多，不胜枚举的去中心化应用(Dapp)。

![Untitled](assets/intro/Untitled.png)

2. 合约的组成部分
